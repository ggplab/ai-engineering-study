# Contributing Guide

## 문서 작성 원칙
- 한 단원 = 한 파일
- 질문/답글/모범정답/배운점 섹션 유지
- 문장은 짧고, 핵심 위주로 작성

## 커밋 메시지 규칙
- `docs: add chapter XX topic`
- `docs: revise chapter XX answers`
- `chore: restructure folders`

## 리뷰 체크리스트
- 질문이 학습 목표와 연결되어 있는가?
- 답글에 근거(개념/예시)가 포함되어 있는가?
- 모범정답이 오해 포인트를 명확히 정리했는가?
- 다음 학습 액션이 기록되어 있는가?
